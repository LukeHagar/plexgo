# PostUsersSignInDataAutoSelectSubtitle

The auto-select subtitle mode (0 = Manually selected, 1 = Shown with foreign audio, 2 = Always enabled)

| `PostUsersSignInDataAutoSelectSubtitleDisable` | 0                                              |
