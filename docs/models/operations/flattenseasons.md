# FlattenSeasons

Setting that indicates if seasons are set to hidden for the show. (-1 = Library default, 0 = Hide, 1 = Show).



## Values

| Name                           | Value                          |
| ------------------------------ | ------------------------------ |
| `FlattenSeasonsLibraryDefault` | -1                             |
| `FlattenSeasonsHide`           | 0                              |
| `FlattenSeasonsShow`           | 1                              |
